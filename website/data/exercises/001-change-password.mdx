1) Ga naar het `Gebruikers en groepen` menu (zie screenshot hieronder). Je kan het startdialoog openen via het Linux Mint logo in de linker onderhoek van het scherm

![Open user screen](/static/images/tutorial-change-password/open-user-screen.png)

Vooraleer je het venster kan openen zal je waarschijnlijk jouw wachtwoord moeten ingeven.

2) Selecteer je gebruikersaccount in de lijst en klik op het paswoord veld. In het voorbeeld hieronder willen we het wachtwoord wijzigen van de gebruiker `tester`

![Open user screen](/static/images/tutorial-change-password/click-password.png)

3) Nu kan je je paswoord wijzigen in het dialoog dat te voorschijn komt.

![Open user screen](/static/images/tutorial-change-password/change-password.png)

> Tip: Kies je paswoord niet te kort, zodat anderen het niet gemakkelijk kunnen achterhalen.
