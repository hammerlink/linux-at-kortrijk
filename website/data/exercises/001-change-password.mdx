1) Ga naar het `Gebruikers en groepen` menu (zie screenshot hieronder). Je kan het startdialoog openen via het Linux Mint logo in de linker onderhoek van het scherm

<img src="/static/images/tutorial-change-password/open-user-screen.png" alt="Open User Screen" style={{maxWidth:'400px', width:'100%'}} />

Vooraleer je het venster kan openen zal je waarschijnlijk jouw wachtwoord moeten ingeven.

2) Selecteer je gebruikersaccount in de lijst en klik op het paswoord veld. In het voorbeeld hieronder willen we het wachtwoord wijzigen van de gebruiker `tester`

<img src="/static/images/tutorial-change-password/click-password.png" alt="Click Password" style={{maxWidth:'400px', width:'100%'}} />

3) Nu kan je je paswoord wijzigen in het dialoog dat te voorschijn komt.

<img src="/static/images/tutorial-change-password/change-password.png" alt="Change Password" style={{maxWidth:'400px', width:'100%'}} />

Voor tips omtrent hoe je een goed paswoord kan kiezen, kan je naar [deze link gaan](https://www.dnsbelgium.be/nl/hoe-kies-ik-een-sterk-wachtwoord-dat-ik-niet-vergeet)
