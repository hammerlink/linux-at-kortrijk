title: "Oefening 9: Software installeren en verwijderen"
summary: "Programma's installeren, gebruiken en verwijderen via de Software Manager in Linux Mint."
draft: false
date: 2025-09-09
---

1) **Software Manager openen**  
   - Klik linksonder op het **menu**.  
   - Ga naar **Beheer → Softwarebeheer** (Software Manager).  

2) **Zoek naar een programma**  
   - Typ in de zoekbalk: **Mijnenveger**.  
   - Je krijgt een lijst met beschikbare versies (bv. GNOME Mines).  

3) **Installeren**  
   - Klik op het programma **Mijnenveger**.  
   - Klik op **Installeren**.  
   - Voer je wachtwoord in om de installatie te bevestigen.  

4) **Spelen**  
   - Na de installatie klik je in Software Manager op **Starten**,  
     of open het spel via het **menu → Games → Mijnenveger**.  
   - Speel even een potje!  

5) **Verwijderen**  
   - Ga opnieuw naar **Software Manager**.  
   - Zoek naar **Mijnenveger**.  
   - Klik op **Verwijderen**.  
   - Bevestig indien gevraagd met je wachtwoord.  

6) Controleer dat het spel verdwenen is uit het **menu → Games**.
