# Linux en back-ups

Eén van de grootste problemen die mensen tegenkomen bij het overstappen van windows naar Linux, is het vinden van een degelijk back-up systeem. Dit systeem mag enerzijds niet te moeilijk zijn in gebruik, maar moet anderzijds ook betrouwbaar zijn. Je wil er immers zeker van zijn dat afbeeldingen, documenten, ... niet verloren gaan als jouw PC het begeeft.

## Windows en OneDrive

Hier heeft Windows een stap(je) voor, gezien er standaard OneDrive wordt meegeleverd op ieder systeem. Je krijgt 5Gb opslag gratis. Als je meer geheugen wil, moet je je een abonnement aanschaffen (zie ook [Handige links](#handige-links)).
Persoonlijk vind ik de standaard OneDrive installatie op een Windows machine niet ideaal, met de volgende redenen:

- **Standaard krijg je een probeer versie:** Het gratis pakket biedt 5Gb opslag aan. In de realiteit loop je al snel tegen deze limiet aan, waarna een (niet erg goedkoop) abonnement noodzakelijk is. OneDrive brengt dus verborgen kosten met zich mee: eenmaal je het gewoon bent om ermee te werken, loop je tegen de geheugen limiet aan en heb je een abonnement nodig. Bij Apple heb je gelijkaardige prakijken met iCloud.
- **Het is niet compatibel met Linux:** er bestaat geen officiële Linux client. Je kan echter wel files gaan uploaden/download op Linux via de Web client

OneDrive heeft echter ook een aantal onmiskenbare voordelen:

- Mooi geïntegreerd in de besturingssystemen waar het wel ondersteund wordt (Windows, MacOS, iOS en Android)
- Eenvoudig in gebruik
- Gebundelde Office producten zijn wijd verspreid - iedereen kan er dus mee werken
- Handige tools om samen te werken met anderen (je kan links naar bestanden delen, samenwerken in documenten via Word of Excel, ...)
