# Linux en back-ups

Een veelgebruikte functionaliteit van PC gebruikers is het maken van back ups in de cloud. Hoewel dit zeker geen absolute vereiste is, kan het wel handig zijn om bepaalde files toch te gaan synchronizeren naar een extern systeem, ik denk maar aan foto's of belangrijke documenten.

## Windows en OneDrive

Bij het maken van back-ups heeft Windows een stap(je) voor, gezien er standaard OneDrive wordt meegeleverd op ieder systeem [sinds windows 10](https://support.microsoft.com/en-us/office/onedrive-desktop-app-for-earlier-versions-of-windows-850703dd-ea56-4c7a-bff5-6c2e4da227cf). Je krijgt 5Gb opslag gratis. Als je meer geheugen wil, moet je je een abonnement aanschaffen (zie ook [Handige links](#handige-links)).
Persoonlijk vind ik de standaard OneDrive installatie op een Windows machine niet ideaal, met de volgende redenen:

- **Standaard krijg je een zeer beperkte versie:** Het gratis pakket biedt 5Gb opslag aan. In de realiteit loop je al snel tegen deze limiet aan, waarna een (niet erg goedkoop) abonnement noodzakelijk is. OneDrive brengt dus verborgen kosten met zich mee: eenmaal je het gewoon bent om ermee te werken, loop je tegen de geheugen limiet aan en heb je een abonnement nodig. Bij Apple heb je gelijkaardige prakijken met iCloud.
- **Het is niet compatibel met Linux:** er bestaat geen officiële Linux client. Je kan echter wel files gaan uploaden/downloaden op Linux via de Web client

OneDrive heeft echter ook een aantal onmiskenbare voordelen:

- **Mooi geïntegreerd** in de besturingssystemen waar het wel ondersteund wordt (Windows, MacOS, iOS en Android)
- Eenvoudig in gebruik
- **Gebundelde Office producten** zijn wijd verspreid - iedereen kan er dus mee werken
- Handige **tools om samen te werken** met anderen (je kan links naar bestanden delen, samenwerken in documenten via Word of Excel, ...)

## Linux-vriendelijke alternatieven

### Geen cloud nodig?

Hoewel cloudopslag zeker handig kan zijn, is het goed om te beseffen dat je **niet per se een cloud back-up nodig hebt**. Voor veel gebruikers is een **eenvoudige lokale opslag** voldoende. Door je foto's en documenten op een gestructureerde manier op je eigen harde schijf op te slaan, kan je al heel wat overzicht en veiligheid creëren.

Zolang je regelmatig manueel een kopie maakt van je belangrijkste bestanden op een externe harde schijf of USB-stick, voldoe je eigenlijk al aan de basisprincipes van een goede back-upstrategie. Deze aanpak heeft bovendien als voordeel dat je **volledig controle hebt over je gegevens**, zonder afhankelijk te zijn van externe diensten of abonnementen.
