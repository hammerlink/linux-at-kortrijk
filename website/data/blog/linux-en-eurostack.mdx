# Wat is de link tussen Linux en EuroStack?

## Wat is Eurostack en waarom bestaat het?

Europa hinkt achterop als het gaat om de **digitale revolutie van de laatse decennia**. Als ik u vraag: noem mij 5 grote tech bedrijven op, is de kans groot dat 4 van die 5 bedrijven die u mij opnoemt **Amerikaanse** spelers zijn. Dit achterop lopen wordt door [het rapport van Mario Draghi](https://commission.europa.eu/topics/eu-competitiveness/draghi-report_en) (voormalig president van de Europese Centrale Bank) gezien als een van de hoofdzaken voor de **slenterende Europese economie**.

De EuroStack is een **initiatief van de Europese Unie** om hier iets aan te doen. Het initiatief vraagt om minder afhankelijkheid van andere grootmachten als het gaat om de technologiesector. De technologiesector wordt hier als heel breed begrip genomen, gaande van low-level chip industrieen tot high-level AI toepassingen.

## Wat heeft de EuroStack met linux te maken?

Als het gaat om besturingssystemen voor PCs, bestaan er slechts 3 grote spelers:

- Microsoft Windows
- MacOS
- Linux Desktop

Windows en MacOS zijn afkomstig van grote Amerikaanse tech bedrijven en zijn **propriëtair**. Linux Desktop daarentegen is een vrij te gebruiken besturingssysteem, waarvan de broncode voor het grootste gedeelte openbaar gemaakt is.
Het spreekt dus vanzelf dat voor het opbouwen van een **open Europese technologie stack** de eerste 2 besturingssystemen geen optie zijn. Daarom wordt eigenlijk uitsluitend naar de 3de mogelijkheid gekeken voor het opbouwen van deze "EuroStack".

Hier moet echter een kanttekening gemaakt worden: **er is niet één Linux Desktop besturingssyteem**. Er is keuze uit heel wat verschillende opties, distro's genoemd. De verschillende opties zijn telkens gericht op een specifiek doelpubliek: van een eenvoudig systeem om de dagelijkse werkjes (web browsen, mails sturen, ...) te kunnen doen, tot meer complexe systemen voor doorgewinterde programmeurs of andere professionals. Er is voor elk wat wils in de Linux wereld. Dit staat in contrast met Microsoft Windows en MacOS, die slechts een gelimiteerde set van besturingssystemen aanbieden, waardoor de moduleerbaarheid beperkt is.

De **interesse van de EuroStack beweging in het Linux Desktop systeem** is niet alleen te wijten aan het feit dat dit het enige systeem is dat niet gelinkt is aan een groot Amerikaans Tech bedrijf. Zoals ik eerder reeds vermeldde, wordt Linux ontwikkeld in een volledig open omgeving, waar iedereen toe kan bijdragen. Dit systeem vormt dus ook een uitstekende basis voor verdere software ontwikkelingen, ik denk maar aan alternatieven voor Microsoft Office (Word, Outlook enzovoort).
